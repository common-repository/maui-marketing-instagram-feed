== Description ==
Super simple to set up with a quick authentication process thru the browser. Responsive design will fit any container and with the built in shortcode, you can place it almost anywhere.

Click the Instagram image post in the feed to display an overlay with post details such as tags, likes and comments. Only non-private Instagram accounts will work with this plugin.

<h4>Features</h4>
- Super simple to set up Completely responsive and mobile ready - layout looks great on any size screen and in any container width.
- The built-in shortcode allows to completely customize to your Instagram feed.
- View all Instagram photos with up / down button.
- Overlay displays your Instagram photo details such as tags, like or comments.

== Installation ==

1. Install the Instagram plugin either via the WordPress plugin directory, or by uploading the files to your web server (in the `/wp-content/plugins/instagram-maui-marketing).
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'Instagram Maui Marketing' settings page to obtain your Instagram Access Token and User ID and configure your settings.
4. The Use the shortcode `[mm_data_instagram]` in your page, post or widget to display your photos.

For simple step-by-step directions on how to set up the Instagram Maui Marketing plugin please refer to our [setup guide](http://dev.mauimarketing.com/instagram/setup.html)
